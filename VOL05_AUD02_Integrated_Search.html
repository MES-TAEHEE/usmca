01
Overview
화면 개요
Integrated Search gives an auditor one search box across every module at once — certificates, parts, origins, labels and suppliers — returning a single unified, faceted result set so any piece of evidence can be found instantly without hopping between screens. A full-text query runs against a consolidated search index spanning doc_certificate, org_part_origin, lbl_issue and the md_* masters; results are grouped by entity type with icons, status badges and key fields. A left facet panel (module / period / status / country) narrows the set, and every row drills down to its source screen or jumps straight into traceability (AUD-01). Permission-filtered results, access logging (BR-AUD-04) and source-of-truth linkage (BR-AUD-01) make this the auditor's fastest entry point into evidence.
통합 검색은 감사자가 전 모듈을 한 번에 검색하는 단일 검색창입니다. 증빙·부품·원산지·라벨·공급사를 가로질러 통합·패싯 결과를 반환해, 화면 이동 없이 어떤 증적이든 즉시 찾습니다. 전문(full-text) 질의가 doc_certificate·org_part_origin·lbl_issue·md_* 마스터를 아우르는 통합 검색 인덱스를 조회하며, 결과는 엔티티 유형별(아이콘·상태배지·핵심필드)로 그룹화됩니다. 좌측 패싯 패널(모듈/기간/상태/국가)로 좁히고, 각 행은 원본 화면 열람 또는 역추적(AUD-01)으로 바로 이동합니다. 권한 필터·접근 로그(BR-AUD-04)·SoT 연결(BR-AUD-01)로 증적 진입의 최단 경로를 제공합니다.

EXC-01No Results결과 없음
Trigger질의·패싯 조합에 일치 항목 없음.
Action빈 상태 + 유사어·패싯 완화 제안 표시.
검색 결과가 없습니다 · 필터를 완화해 보세요
EXC-02Too-Broad Query검색 범위 과다
Trigger결과가 임계치(예: 1,000건) 초과.
Action상위 N건만 표시 + 패싯으로 좁히기 안내.
결과가 너무 많습니다 — 필터로 좁혀 주세요
EXC-03Permission-Filtered권한 필터
Trigger사용자 권한 외 모듈·항목 포함 질의.

Action"최신화 N분 전" 배지 + [새로고침] 제공.

TriggerExcel 출력 건수가 즉시 한도 초과.

07
Business Rules
업무 규칙
IDRule · 규칙Action
BR-AUD-04
Access logged & permission-scoped — every search, view, and export is logged (who / when / query); results are filtered to the user's RBAC scope.
모든 검색·열람·내보내기 로그 기록(누가/언제/무엇). 결과는 사용자 권한(RBAC) 범위로 필터.
자동
BR-AUD-01
Source-of-truth linkage — every result row points to its live source record; search never shows stale copies, only references to current SoT.
모든 결과 행은 실시간 원본(SoT)을 가리킴. 사본·스냅샷이 아닌 현행 원본 참조만 표시.
차단
BR-AUD-05
Gaps surfaced — missing or expired evidence (no certificate, expired COO) is flagged in results so auditors see compliance gaps, not just hits.
누락·만료 증적(증빙 없음, COO 만료 등)을 결과에 표시 — 단순 검색결과가 아닌 컴플라이언스 공백까지 노출.
