01
Overview
화면 개요
BOM Origin Rollup rolls component origins up the multi-level BOM to determine the finished-good (FG) origin and to assemble the judgment basis (원산지 판정 근거) the program calls for. The screen pulls the structured BOM from INT (SAP), joins each leaf component to its registered origin from ORG-01, and applies the determination rules (substantial transformation / tariff shift) node by node. The finished-good origin is the rollup result (BR-ORG-05). Any part lacking a registered origin or missing BOM linkage forces the whole result into "보류 / Pending" (BR-ORG-03) — the system never guesses. Every node, its origin flag, and the rule applied are captured as the auditable evidence trail (BR-ORG-01) feeding ORG-05 (RVC), the COO label (LBL), and the audit dossier (AUD).
BOM 원산지 집계는 부품 원산지를 다단계 BOM을 따라 상향 집계하여 완제품 원산지를 판정하고, 프로그램이 요구하는 원산지 판정 근거를 구성합니다. INT(SAP)에서 BOM 구조를 가져와 각 말단 부품을 ORG-01의 등록 원산지와 결합하고, 노드별로 판정 규칙(실질적 변형/세번변경)을 적용합니다. 완제품 원산지는 집계 결과입니다(BR-ORG-05). 원산지 미등록 또는 BOM 미연결 부품이 하나라도 있으면 결과는 "보류" 상태가 됩니다(BR-ORG-03) — 시스템은 추정하지 않습니다. 모든 노드·원산지 플래그·적용 규칙은 감사 가능한 근거(BR-ORG-01)로 기록되어 ORG-05(RVC)·COO 라벨(LBL)·감사 자료(AUD)로 연계됩니다.

Exception Handling
예외 처리
EXC-01Missing Component Origin부품 원산지 미등록
Trigger말단 노드가 ORG-01에 등록 원산지 없음.
Action결과 "보류" 처리(BR-ORG-03). 해당 노드 노랑 표시 + ORG-01 등록 링크.
원산지 미등록 부품 — 결과 보류 (Pending)
EXC-02BOM Not Linked (INT)BOM 미연결
TriggerINT(SAP)에 해당 자재의 BOM 구성이 없음.
Action노드 빨강 표시. 집계 미완 → "보류". INT 동기화 안내.
BOM 미연결 — INT에서 구성 확인 필요
EXC-03Circular BOM순환 참조
TriggerBOM 전개 중 노드가 자기 상위를 재참조(루프).
Action전개 중단 + 순환 경로 표시. 집계 차단.
순환 BOM 감지 — 구조 검토 필요
EXC-04Cost Weight Missing단가비중 누락
Trigger노드 단가/비중 정보 없음.
Action원산지 판정은 진행하되 RVC 영향 경고. ORG-05 계산 정확도 저하 안내.
단가비중 누락 — RVC 결과 영향 가능
EXC-05Stale BOM VersionBOM 버전 경과
Trigger집계에 사용한 BOM Rev가 INT 최신과 불일치.
Action경고 배너 + "INT 재동기화" 권유. 이전 Rev 결과는 참조용 표시.
최신 BOM 아님 — 재동기화 권장
EXC-06No Permission권한 없음
TriggerORG.View만 보유 — 실행 권한 없음.
Action[집계 실행] 비활성. 기존 결과 열람만 가능.
집계 실행 권한이 없습니다 (ORG.Run 필요)

07
Business Rules
업무 규칙
IDRule · 규칙Action
BR-ORG-03
Pending on gaps — if any component lacks a registered origin or its BOM is not linked, the rollup result is held as "Pending"; FG origin cannot be confirmed until resolved. No guessing.
부품 원산지 미등록 또는 BOM 미연결 시 결과는 "보류". 해소 전까지 완제품 원산지 확정 불가 — 추정 없음.
차단
BR-ORG-05
FG = rollup result — the finished-good origin is determined solely by rolling component origins up the BOM with the applied determination rules; it is never entered manually.
완제품 원산지는 부품 원산지의 BOM 상향 집계 + 판정 규칙 적용 결과로만 결정. 수기 입력 불가.
자동
BR-ORG-01
Evidence trail — every node, its origin flag, and the rule applied are stored as the auditable judgment basis (원산지 판정 근거) for ORG-05, LBL, and AUD.
모든 노드·원산지 플래그·적용 규칙을 감사 가능한 원산지 판정 근거로 저장(ORG-05·LBL·AUD 연계).
