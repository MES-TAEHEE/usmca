ORIGIN
MANAGEMENT
원산지 관리 — 부품·완제품 원산지 등록 · 국가별 추적 · 변경 이력 · BOM 원산지 집계 · RVC 자동판정
The Phase 3 origin engine. Registers the country of origin for every part and finished good together with its determination basis and supporting evidence (sourced from DOC). Tracks origin by country across the 4-country supply chain (KR / US / MX / CN), keeps an immutable change audit log, and rolls component origins up through the BOM to determine the finished-good origin (with judgment basis). Computes RVC for USMCA preference (Phase 4). Consumes DOC (evidence) and INT (BOM / SAP).
Phase 3 원산지 엔진. 모든 부품·완제품의 원산지국을 판정 근거 및 증빙(DOC 연계)과 함께 등록합니다. 4개국 공급망(한국 / 미국 / 멕시코 / 중국) 전반의 국가별 원산지를 추적하고, 불변 변경 감사로그를 유지하며, 구성품 원산지를 BOM을 따라 롤업하여 완제품 원산지를 판정(판정 근거 포함)합니다. USMCA 특혜용 RVC를 자동 계산(Phase 4). DOC(증빙)·INT(BOM / SAP)로 연계됩니다.

EN · Description
Central registry of country of origin for each part and finished good. Grid lists part no., description, supplier, origin country, determination basis (WO / Tariff Shift / RVC), linked DOC evidence, and status. Registration requires the determination basis and a valid DOC certificate (BR-ORG-01). Feeds country tracking (ORG-02) and BOM rollup (ORG-04); every edit writes an audit log (ORG-03).

Country-level view of origin across the KR / US / MX / CN supply chain. Aggregates parts and value by origin country, surfaces non-USMCA (e.g. CN) exposure, and flags parts still pending origin. Supports drill-down to ORG-01 by country. ISO 3166 codes are enforced (BR-ORG-06).
KO · 설명
한국/미국/멕시코/중국 공급망 전반의 국가별 원산지 뷰. 원산지국별 부품·금액을 집계하고, 비(非)USMCA(예: 중국) 노출을 부각하며, 원산지 미등록 부품을 플래그. 국가별 ORG-01 드릴다운 지원. ISO 3166 국가코드 강제(BR-ORG-06).

ORG-04 · ⚙ CORE
BOM Origin Rollup
BOM 원산지 집계 — 구성품 → 완제품 원산지 판정
🖥 Web PC⚙ Core
📋 L3 Detail →

What you'll see: BOM 트리 + 구성품별 원산지·금액 롤업 + 완제품 원산지 판정 결과(판정 근거) + 미연계 부품 보류 표시모듈 핵심 — 구성품 원산지를 BOM 따라 집계해 완제품 원산지를 결정
EN · Description
The module's core engine. Walks the BOM (from INT/SAP) and rolls component origins and values up to determine the finished-good origin, showing the judgment basis (WO / Tariff Shift / RVC). Parts with no BOM linkage put the finished good's determination on hold (BR-ORG-03). The finished-good origin is derived strictly from the rollup (BR-ORG-05) and feeds RVC (ORG-05).
KO · 설명
모듈 핵심 엔진. BOM(INT/SAP)을 순회하며 구성품 원산지·금액을 롤업해 완제품 원산지를 판정하고 판정 근거(완전생산/세번변경/RVC)를 표시. BOM 미연계 부품은 완제품 판정을 보류(BR-ORG-03). 완제품 원산지는 롤업에 근거해서만 도출(BR-ORG-05)되며 RVC(ORG-05)로 연계.

ORG-05
RVC Calculation
RVC 자동판정 — USMCA 특혜 역내 부가가치 (Phase 4)
🖥 Web PCPhase 4
📋 L3 Detail →
👀
What you'll see: RVC 계산기(Net Cost / Transaction Value) + 임계값 대비 통과/미달 + 자동차 부품 임계 예시 + 특혜 적용 판정(Phase 4) BOM 롤업 결과를 기반으로 USMCA 특혜 RVC 자동 산출
EN · Description
Computes Regional Value Content for USMCA preference (Phase 4) using either the Net Cost or Transaction Value method, driven by the ORG-04 rollup and INT cost data. Compares against product-specific thresholds (e.g. automotive parts) and flags pass/fail. Below-threshold results raise a warning (BR-ORG-04) for review before claiming preference.
KO · 설명
USMCA 특혜용 역내 부가가치(RVC)를 순원가법(Net Cost) 또는 거래가격법(Transaction Value)으로 자동 산출(Phase 4). ORG-04 롤업과 INT 원가 데이터를 사용. 품목별 임계값(예: 자동차 부품) 대비 통과/미달 판정. 임계 미달 시 경고(BR-ORG-04) — 특혜 적용 전 검토 필요.

▼ RVC METHODS · RVC 계산 방식
MethodFormula · 산식Description · 설명Threshold · 임계(예)
Net Cost (NC)RVC = (NC − VNM) / NC × 100순원가법. 순원가에서 비원산지재료가치(VNM) 차감. 자동차 분야 주 방식.승용차 75%
Transaction Value (TV)RVC = (TV − VNM) / TV × 100거래가격법. 거래가격 기준 산정. 일반적으로 임계값이 더 높음.일반 ≥ NC
Core PartsRVC (Net Cost)자동차 핵심부품(엔진·변속기 등). 단계적 상향 임계.75%
Other PartsRVC (NC / TV)주요·기타 자동차 부품. 부품군별 차등 임계.65% / 70%
