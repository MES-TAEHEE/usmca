SUPPORT
감사 대응 — CBP 검증·OEM 정기감사 대응 · 부품→증빙→완제품 역추적 · 통합 검색 · 리포트·증빙 패키지 자동 출력
The cross-cutting audit layer. When CBP requests verification or an OEM runs a periodic audit, AUD turns scattered evidence into a defensible response in minutes. Traceability walks the chain parts → supporting evidence → finished goods; Integrated Search spans every module at once; the Report Generator auto-produces origin / part-evidence / RVC-basis reports; and Package Export bundles a complete, validated evidence package. Consumes DOC (certificates), ORG (origin determinations), and LBL (issued labels) data. Primary goal — drastically cut audit response time.
전 모듈을 가로지르는 감사 대응 계층. CBP 검증 요청이나 OEM 정기감사 시, 분산된 증빙을 수 분 내 방어 가능한 답변으로 전환합니다. 역추적(Traceability)은 부품 → 증빙 → 완제품 체인을 추적하고, 통합 검색은 모든 모듈을 한 번에 조회하며, 리포트 생성기는 원산지·부품별 증빙·RVC 근거 리포트를 자동 생성, 패키지 출력은 검증된 완전한 증빙 패키지를 묶어냅니다. DOC(증빙)·ORG(원산지 판정)·LBL(발행 라벨) 데이터를 활용합니다. 핵심 목표 — 감사 대응 시간 대폭 단축.

EN · Description
Visual chain explorer linking parts ↔ supporting evidence (DOC) ↔ origin determinations (ORG) ↔ finished goods & labels (LBL). Bidirectional drill answers both "what evidence backs this part?" and "which parts feed this finished good?". Built on immutable source data — never re-derived. Missing or expired evidence is highlighted in red.
KO · 설명
부품 ↔ 증빙(DOC) ↔ 원산지 판정(ORG) ↔ 완제품·라벨(LBL)을 잇는 시각 체인 탐색기. 양방향 드릴로 "이 부품의 근거 증빙은?", "이 완제품에 들어간 부품은?"을 모두 답변. 불변 원천 데이터 기반 — 재계산하지 않음. 누락·만료 증빙은 빨강 강조.

AUD-02
Integrated Search
통합 검색 — 전 모듈 단일 검색창
🖥 Web PCPhase 1~
📋 L3 Detail →
👀
What you'll see: 단일 검색창 + 모듈별(DOC/ORG/LBL) 결과 패싯 + 키워드·식별자·기간·공급사 필터감사관이 던지는 어떤 키로도 즉시 원천 데이터에 도달
EN · Description
One search box across every module at once. Query by part no., certificate no., supplier, HTS, label/lot, or free text; results are faceted by module (DOC / ORG / LBL) with deep links into the source record and into AUD-01 traceability. Eliminates module-hopping during live audits.
KO · 설명
전 모듈을 한 번에 검색하는 단일 창구. 부품번호·인증서번호·공급사·HTS·라벨/LOT·자유 텍스트로 조회, 결과를 모듈별(DOC/ORG/LBL) 패싯으로 제공하고 원천 레코드 및 AUD-01 역추적으로 딥링크. 실시간 감사 중 모듈 이동 제거.

AUD-03
Report Generator
감사 리포트 생성 — 템플릿 기반 자동 작성
🖥 Web PCPhase 1~
📋 L3 Detail →
👀
What you'll see: 리포트 템플릿 선택(원산지 증명·부품별 증빙·RVC 근거) + 대상 범위 지정 + 자동 채움 미리보기 + 생성 이력반복 수작업 없이 감사 리포트를 자동 작성
EN · Description
Template-driven report builder. Pick a template (Origin Certification Package / Part-by-Part Evidence / RVC Basis), set scope, and the system auto-populates from DOC/ORG/LBL source data. Preview before generating; every generation is logged with watermark/hash for tamper-evidence. Drives the package export (AUD-04).
KO · 설명
템플릿 기반 리포트 생성기. 템플릿(원산지 증명 패키지·부품별 증빙·RVC 근거)을 선택하고 범위를 지정하면 DOC/ORG/LBL 원천 데이터에서 자동 채움. 생성 전 미리보기; 모든 생성은 워터마크/해시와 함께 이력 기록(변조 방지). 패키지 출력(AUD-04)으로 연계.

AUD-04
Audit Package Export
증빙 패키지 출력 — 완전한 제출본 묶음
🖥 Web PCPhase 1~
📋 L3 Detail →
👀
What you'll see: 패키지 구성 항목 체크리스트 + 유효성 검증(만료/누락 경고) + 목차·색인 자동 생성 + ZIP/PDF 출력제출 직전 누락·만료 증빙을 차단하여 불완전 제출 방지
EN · Description
Bundles a complete, submission-ready evidence package (reports + underlying certificates + traceability snapshot) into a single ZIP/PDF with auto index. A validation pass includes only valid evidence and warns on expired/missing items before export, preventing incomplete submissions. Output is watermarked/hashed for integrity.
KO · 설명
완전한 제출용 증빙 패키지(리포트 + 근거 인증서 + 역추적 스냅샷)를 목차 자동 생성과 함께 단일 ZIP/PDF로 묶음. 유효성 검증으로 유효 증빙만 포함하고 출력 전 만료·누락 항목을 경고하여 불완전 제출 방지. 출력물은 무결성을 위해 워터마크/해시 처리.
