01
Overview
화면 개요
The Certificate Viewer is a split-view review screen: the left pane renders the original PDF / image of the certificate (zoom, rotate, page navigation), while the right pane presents all metadata and linked entities — cert no., type, supplier, origin country, issue / expiry, linked parts, version history, and the full approval trail. Validity status and remaining days are shown prominently at the top so an auditor can judge at a glance whether the evidence is currently valid. Entry is from a DOC-01 grid row click. Every view and download is written to the audit log (BR-DOC-06), and only approved documents are surfaced as valid evidence (BR-DOC-02). The goal: review evidence together with its full context on one screen, ready for audit.
증빙 뷰어는 분할 화면 열람 화면입니다. 좌측에는 증빙 원본 PDF/이미지가 렌더링되며(확대·회전·페이지 이동), 우측에는 모든 메타데이터와 연계 엔터티(인증서번호·유형·공급사·원산지국·발급/만료일·연계 품목·버전 이력·승인 트레일)가 표시됩니다. 유효 상태와 잔여 일수는 상단에 강조 표시되어 감사자가 한눈에 현재 유효성을 판단할 수 있습니다. DOC-01 그리드 행 클릭으로 진입하며, 모든 조회·다운로드는 감사 로그에 기록되고(BR-DOC-06), 승인된 증빙만 유효 증빙으로 노출됩니다(BR-DOC-02). 목적: 증빙을 전체 컨텍스트와 함께 한 화면에서 검토하여 감사 대응을 준비하는 것입니다.

Exception Handling
예외 처리
EXC-01File Missing / Corrupt파일 누락·손상
Trigger원본 파일 미존재 또는 렌더링 실패.
Action미리보기 폴백 표시. 메타데이터는 정상 노출. 관리자 재업로드 안내.
미리보기를 불러올 수 없습니다 · File unavailable
EXC-02Expired Certificate만료된 증빙
Trigger만료일 경과 (D-Day < 0).
Action상단 경고 배너 + 빨강 배지. 유효 증빙으로 인정 불가 안내(BR-DOC-02).
⚠ 만료된 증빙 — 갱신 필요 · Expired
EXC-03No View Permission열람 권한 없음
TriggerDOC.View 미보유 사용자 접근.
Action뷰어 진입 차단. 권한 안내 후 목록으로 복귀.
열람 권한이 없습니다 (DOC.View 필요)
EXC-04Large File대용량 파일
Trigger고용량 PDF/이미지 로드.
Action청크 스트리밍 + 페이지별 지연 로드. 진행 표시기 노출.
미리보기를 불러오는 중… · Streaming
EXC-05Superseded Version대체된 버전
Trigger최신이 아닌 이전 버전 열람.
Action"대체됨" 알림 배너 + 최신 버전 이동 링크(DOC-05).
이 버전은 대체되었습니다 — 최신 버전 보기
EXC-06Download Blocked다운로드 차단
Trigger미승인(검토중) 증빙 다운로드 시도.
Action다운로드 차단(BR-DOC-02). 승인 후 가능 안내.
승인 전 증빙은 다운로드할 수 없습니다

07
Business Rules
업무 규칙
IDRule · 규칙Action
BR-DOC-06
Audit trail — every view and download in the viewer is logged (who / when / what) for audit retention.
뷰어의 모든 조회·다운로드는 감사 보존을 위해 전수 로그 기록(누가·언제·무엇).
자동
BR-DOC-02
Only approved shown as valid — non-approved (Pending Review) certificates are not surfaced as valid evidence and cannot be downloaded.
승인된 증빙만 유효로 표시. 미승인(검토중) 증빙은 유효 증빙으로 노출되지 않으며 다운로드 불가.
차단
BR-DOC-05
Immutable versions — viewed versions are read-only and immutable; renewals create new versions, prior versions preserved.
열람 버전은 읽기 전용·불변. 갱신은 신규 버전 생성, 이전 버전 보존.
