01
Overview
화면 개요
Country of Origin Tracking aggregates and monitors declared origin across the four-country supply chain (🇰🇷 KR / 🇺🇸 US / 🇲🇽 MX / 🇨🇳 CN). The screen rolls up part-level origin records into a country view, showing how many parts and suppliers map to each country of origin and the count of finished goods affected. It surfaces non-USMCA-originating content (e.g. CN) that erodes preference eligibility, so compliance can act before it impacts an RVC calculation. Every row drills down to the underlying parts in ORG-01, and the aggregation feeds BOM rollup (ORG-04) and RVC (ORG-05). Data is read-only here — origin is maintained in ORG-01.
국가별 원산지 추적은 4개국 공급망(🇰🇷 KR / 🇺🇸 US / 🇲🇽 MX / 🇨🇳 CN) 전반의 신고 원산지를 집계·모니터링합니다. 부품 단위 원산지 데이터를 국가별로 롤업하여 각 원산지국에 매핑된 부품 수·공급사 수와 영향받는 완제품 수를 보여줍니다. 특혜 자격을 저하시키는 비USMCA 원산지(예: CN) 비중을 부각해, RVC 산정 영향 전에 컴플라이언스가 선제 대응하도록 합니다. 각 행은 ORG-01 부품 목록으로 드릴다운되며, 집계 결과는 BOM 롤업(ORG-04)·RVC(ORG-05)에 연계됩니다. 본 화면은 조회 전용이며 원산지 입력은 ORG-01에서 수행합니다.

05
Exception Handling
예외 처리
EXC-01Parts With No Origin원산지 미입력 부품
TriggerORG-01에 원산지가 입력되지 않은 부품 존재.
Action"미지정(Pending)" 그룹으로 별도 집계 + 카운트 표시. ORG-01 입력 안내.
원산지 미입력 부품 N건 — ORG-01에서 지정 필요
EXC-02Mixed Origin Part복수 원산지 부품
Trigger동일 부품이 복수 공급사·복수 원산지로 등록됨.
Action각 원산지에 분할 카운트 + "혼합 원산지" 주석. 중복 합산 방지.
복수 원산지 — 공급사별로 분리 집계됨
EXC-03Invalid Country Code국가코드 오류
TriggerISO 미준수·미정의 국가코드 데이터(BR-ORG-06).
Action"Unknown" 버킷 분리 표시. 데이터 정정 대상 플래그.
유효하지 않은 국가코드 — ISO 매핑 확인 필요
EXC-04Stale Aggregation집계 캐시 만료
Trigger집계 캐시가 최신 원산지 변경을 반영하지 못함.
Action"오래된 데이터" 배너 + [새로고침]으로 재집계 트리거.
집계 기준이 오래되었습니다 — 새로고침
EXC-05Large Export대용량 내보내기
TriggerExport 대상 행이 임계치 초과.
Action비동기 생성 후 다운로드 링크 알림. 화면 차단 없음.
파일 생성 중 — 완료 시 알림드립니다
EXC-06No Permission권한 없음
TriggerORG.View 미보유 사용자 접근.
Action화면 접근 차단. 권한 요청 안내.
조회 권한이 없습니다 (ORG.View 필요)

07
Business Rules
업무 규칙
IDRule · 규칙Action
BR-ORG-06
ISO country mapping — every origin must resolve to a valid ISO 3166 country code; USMCA-originating set = {US, MX, CA}, all others (incl. KR, CN) are non-originating for preference.
모든 원산지는 유효 ISO 3166 국가코드로 매핑. USMCA 원산지 집합={US, MX, CA}, 그 외(KR·CN 포함)는 특혜상 非원산지.
자동
BR-ORG-05
Finished-good origin from rollup — country-level impact and finished-good origin are derived from BOM rollup (ORG-04), not declared directly on this screen.
완제품 원산지·영향은 본 화면이 아닌 BOM 롤업(ORG-04) 결과에서 도출.
자동
BR-ORG-01
Evidence-backed — each part's declared origin must be supported by registered evidence (DOC); origins without evidence are excluded from USMCA-originating counts.
각 부품 원산지는 등록 증빙(DOC)으로 뒷받침되어야 함. 증빙 없는 원산지는 USMCA 원산지 카운트에서 제외.
