Schema Conventions & Relationships · 스키마 규약 · 관계

PK = <table>_id IDENTITY (BIGINT) unless a natural code key (e.g. md_country.country_code, sys_code_group.group_code). All audit columns: created_by / created_at / updated_by / updated_at. All text is NVARCHAR (Unicode KO/EN); timestamps are UTC via SYSUTCDATETIME(). Two circular references are resolved via ALTER … ADD: md_label_template.oem_id ↔ md_oem_customer and doc_certificate.current_version_id ↔ doc_version. Foreign-key cells below name and link to the referenced table.

Source note · The header comment in UOCM_Schema.sql states 41 tables (SYS 11), but the DDL physically defines 40 CREATE TABLE statements (SYS = 10). This page renders all 40 tables exactly as defined in the SQL; the two ALTER TABLE … ADD CONSTRAINT blocks add circular FKs, not new tables.
