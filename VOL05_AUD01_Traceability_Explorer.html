01
Overview
화면 개요
The Traceability Explorer delivers bi-directional origin traceability. Drilling DOWN from a finished good walks its BOM components (md_bom), each component's declared origin (org_part_origin), the supporting certificate that proves that origin (doc_certificate), and every label issued for it (lbl_issue). Tracing UP from a single certificate reveals all parts and finished goods it affects — the impact set when a cert expires or is revoked. The entire tree is reconstructed live from immutable source data (BR-AUD-01), so any node missing or expired evidence is flagged red (BR-AUD-05). This is the core "재취합 없이 즉시 증빙" capability: instant, defensible audit evidence with no manual re-collection. Selected nodes feed directly into AUD-03 reports and AUD-04 packages.
역추적 탐색기는 양방향 원산지 추적을 제공합니다. 완제품에서 하향(Down) 드릴다운하면 BOM 구성품(md_bom) → 각 구성품의 원산지(org_part_origin) → 그 원산지를 증명하는 인증서(doc_certificate) → 발행된 라벨(lbl_issue)까지 펼쳐집니다. 인증서에서 상향(Up) 추적하면 그 증빙이 영향을 미치는 모든 부품·완제품(만료·폐기 시 영향 범위)을 보여줍니다. 전체 트리는 불변 원천 데이터(BR-AUD-01)로 실시간 재구성되며, 증빙 누락·만료 노드는 빨강으로 강조(BR-AUD-05)됩니다. 이것이 핵심 "재취합 없이 즉시 증빙" 역량입니다. 선택 노드는 AUD-03 리포트·AUD-04 패키지로 바로 연계됩니다.

07
Business Rules
업무 규칙
IDRule · 규칙Action
BR-AUD-01
Immutable source — the trace tree is reconstructed live from source-of-truth tables (md_bom, org_part_origin, doc_certificate, lbl_issue); no manual re-aggregation, no editable copies.
불변 원천 — 추적 트리는 단일 진실원천 테이블에서 실시간 재구성. 재취합·편집본 없음.
자동
BR-AUD-05
Gaps highlighted — any node with missing or expired evidence is flagged red and counted; the tree cannot be presented as complete while gaps exist.
누락 강조 — 증빙 누락·만료 노드는 빨강 강조·카운트. 누락 존재 시 완전 증빙으로 제출 불가.
차단
BR-AUD-03
Valid evidence only — only approved, in-date certificates count as qualifying origin proof in the trace; pending or expired certs are treated as gaps.
유효 증빙만 — 승인·유효기간 내 인증서만 자격 증빙으로 인정. 검토중·만료는 누락 처리.
