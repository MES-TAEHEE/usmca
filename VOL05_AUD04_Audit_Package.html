01
Overview
화면 개요
The Audit Package Export screen assembles a complete, submission-ready evidence package — every certificate, origin record, RVC calculation basis, label, and the generated audit report — into a single indexed ZIP / PDF bundle. Items flow in from AUD-01 (Trace Explorer) and AUD-03 (Audit Report). Only valid evidence is included: expired or missing items are flagged and block export until replaced (BR-AUD-03 / BR-AUD-05). A cover sheet and auto-generated index are prepended, every file is hashed for integrity, and the export action is access-logged with operator, timestamp, and scope (BR-AUD-04). The result is a self-contained, tamper-evident bundle that can be handed directly to a customs auditor.
증빙 패키지 출력 화면은 제출 가능한 완결 증빙 묶음 — 전 인증서·원산지 기록·RVC 산정 근거·라벨·생성된 감사 리포트 — 을 인덱스가 포함된 단일 ZIP / PDF 번들로 조립합니다. 항목은 AUD-01(역추적)·AUD-03(리포트)에서 유입됩니다. 유효한 증빙만 포함되며, 만료·누락 항목은 플래그 처리되어 교체 전까지 출력이 차단됩니다(BR-AUD-03/05). 표지와 자동 인덱스가 선두에 추가되고, 모든 파일은 무결성 해시가 부여되며, 출력 행위는 수행자·시각·범위와 함께 접근 로그로 기록됩니다(BR-AUD-04). 결과물은 세관 감사관에게 즉시 제출 가능한 자기완결·위변조 검증형 번들입니다.

Exception Handling
예외 처리
EXC-01Expired / Missing Item만료·누락 증빙
Trigger검증 시 만료 또는 누락 증빙 포함 (BR-AUD-03 / BR-AUD-05).
Action출력 차단. 해당 행 빨강 강조 + 배너 "N건 만료 — 교체 필요".
유효 증빙으로 교체 후 출력 가능 · Expired/missing evidence
EXC-02Empty Package빈 패키지
Trigger항목 0건 상태에서 검증·출력 시도.
Action검증·출력 버튼 비활성. 최소 1건 담기 안내.
패키지에 항목을 1건 이상 담으세요
EXC-03Large Package (Async)대용량 비동기
Trigger번들 용량이 임계치 초과(예: ≥100MB).
Action비동기 백그라운드 생성. 완료 시 다운로드 링크 통지.
대용량 — 생성 완료 후 알림으로 제공합니다
EXC-04Export Failure출력 실패
Trigger번들 생성 중 파일 접근·병합 오류.
Action트랜잭션 롤백. 부분 산출물 폐기 + 재시도 안내.
번들 생성 실패 — 다시 시도하세요
EXC-05Duplicate Item항목 중복
Trigger이미 담긴 증빙을 재차 담기.
Action중복 경고. 1회만 포함되도록 차단.
이미 패키지에 포함된 항목입니다
EXC-06No Export Permission권한 없음
TriggerAUD.View만 보유한 사용자.
Action출력 버튼 비활성. 열람만 가능.
출력 권한이 없습니다 (AUD.Export 필요)

07
Business Rules
업무 규칙
IDRule · 규칙Action
BR-AUD-03
Valid evidence only — only non-expired, present certificates/records may be exported; validation must pass before ZIP/PDF generation is enabled.
유효(미만료·존재) 증빙만 출력 가능. 검증 통과 전까지 ZIP/PDF 생성 비활성.
차단
BR-AUD-04
Export is logged — every package export records operator, timestamp, package, scope, and file hashes for audit traceability; log is immutable.
출력 행위 전수 기록(수행자·시각·패키지·범위·해시). 로그 수정 불가.
자동
BR-AUD-05
Gaps flagged — expired or missing items are surfaced before export with a clear replace-required prompt; package cannot be finalized while gaps remain.
만료·누락 항목은 출력 전 플래그·교체 안내. 갭 잔존 시 패키지 확정 불가.
